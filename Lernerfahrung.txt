--> es wird benutzt um Objekte als Zeichenketten darzustellen

get/setter Funktionen

das man mit forschleifen effektiver listen ausgeben kann
Bsp:
 def get_all_products(self):
        output = ""
        for produkt in self.liste_produkte:
            output += (produkt) + "\n"
        return output

Umwandlung des Produkts in eine Zeichenkette (str(produkt))

output += str(produkt) + "\n"
Jedes Produkt-Objekt wird mit str(produkt) in eine Zeichenkette umgewandelt.
Die __str__-Methode der Klasse Produkt sorgt dafür, dass die Produktdetails in 
einem lesbaren Format ausgegeben werden.
\n wird hinzugefügt, um jedes Produkt in einer neuen Zeile darzustellen.
